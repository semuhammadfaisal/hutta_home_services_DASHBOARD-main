## 🚀 Quick Reference

### Essential Commands
```bash
# Run migration
cd backend
node migrate-orders-to-payments.js

# Check results
mongosh hutta_home_services --eval "db.payments.countDocuments()"

# Backup database
mongodump --db hutta_home_services --out ./backup

# Restore database
mongorestore --db hutta_home_services ./backup/hutta_home_services
```

### Essential Files
- **Migration Script:** `backend/migrate-orders-to-payments.js`
- **Payment Model:** `backend/models/Payment.js`
- **Orders Route:** `backend/routes/orders.js`
- **Payments Route:** `backend/routes/payments.js`
- **Dashboard Script:** `assets/js/dashboard-script.js`
